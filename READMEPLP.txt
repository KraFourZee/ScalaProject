Scala Programming Language Project Example Programs

*each program was pair programmed

So for these sample programs we wanted to showcase how clean and consise you can have your code be while writing programs in Scala.
Additonally we wanted to show also how scalable the lanugage is. This is where it gets its name Scala. Further we wanted to showcase
some of the differences in the structure of the language such as the for loop and the syntax used for them compared to C++. Also the try catch
statements used also are much more condensed than what java would allow.
